How it works

Physical interception
between agent and action.

Sentinel sits between the agent's decision and real execution. The agent cannot bypass the check — it's architectural, not optional.

01
🤖

Agent decides

Claude, LangChain or your custom agent selects an action to execute.

›
02
🔒

SDK intercepts

Before execution, the decorator calls Sentinel. The action is suspended.

›
03
⚙️

Policy Engine

Your YAML rules evaluate: action type, data class, recipient, context.

›
04
📊

Risk Score

Dynamic 0.0–1.0 score computed from action context in real time.

›
05
🛡️

Decision + PQC Proof

ALLOW / REVIEW / BLOCK — signed with Dilithium-3 NIST FIPS 204.
